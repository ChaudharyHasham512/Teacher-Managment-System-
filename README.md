🎓 Teacher Management System (CRUD Operations) – One Page Web App 

A simple and interactive Teacher Management System built using CRUD (Create, Read, Update, Delete) operations. This project is designed as a single-page application to efficiently manage teacher records with a clean and user-friendly interface.

🚀 Features

➕ Add new teacher details
📋 View all teacher records in a structured layout
✏️ Edit existing teacher information
❌ Delete teacher records instantly
⚡ Real-time UI updates without page reload
🛠️ Technologies Used
HTML5
CSS3 (Styled UI)
JavaScript (DOM Manipulation & CRUD logic)

🎯 Purpose

This project is created for learning and practicing core web development concepts, especially how CRUD operations work in real-world applications. It helps strengthen understanding of dynamic UI handling, data management, and basic frontend logic.

📌 Highlights

One-page responsive design
Beginner-friendly structure
Clean and maintainable code
Ideal for academic or internship submission
